WEB DESIGN

I have the elemenatary knowledge of HTML, CSS, Bootstrap, and JavaScript. I can design and build responsive websites from scratch using these technologies.

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications.

CSS

CSS (Cascading Style Sheets) is a style sheet language used for styling a document written in a markup language like HTML.

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

JavaScript

JavaScript is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It enables interactive web pages and is an essential part of web applications.
